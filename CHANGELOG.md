# Changelog

All notable changes to autoignore are documented here.

---

## [Unreleased]

## [2026-04-06]

### Added

- Interactive template selector (`-i` flag)
- Auto-detection of project type from repository contents
- Shell completions for Bash, Zsh, and Fish
- Templates: Angular, Ansible, Autotools, Bazel, Clojure, Crystal, Dart,
  Django, Emacs, Flutter, Godot, Gradle, Groovy, Julia, Kubernetes, Laravel,
  LaTeX, macOS, Make, Nim, OCaml, Rails, R, Scala, Unreal Engine, Vagrant,
  VSCode, Vue, Windows, Zig

## [2025-06-24]

### Added

- Initial release
- 43 built-in templates covering major languages, frameworks, and IDEs
- `--list` flag to display available templates
- `--output` flag to specify a custom output path
- `--append` flag to append to an existing `.gitignore`
- `--verbose` flag for detailed output
- User template directory (`~/.local/share/autoignore/template/`)
- Colored terminal output
- Apache License 2.0
